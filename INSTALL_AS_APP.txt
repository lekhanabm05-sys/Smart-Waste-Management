│                                                                 │
│  STEP 2: INSTALL ON ANDROID                                    │
│  ═══════════════════════════                                   │
│                                                                 │
│  1. Open Chrome browser on your phone                          │
│  2. Go to: http://YOUR_IP:5000                                 │
│  3. Look for "Install App" button (bottom-right)               │
│  4. Tap it                                                     │
│  5. Confirm installation                                       │
│  6. 🎉 App appears on home screen!                             │
│                                                                 │
│  Alternative:                                                  │
│  - Tap menu (⋮) → "Install app" or "Add to Home screen"       │
│                                                                 │
└─────────────────────────────────────────────────────────────────┘


┌─────────────────────────────────────────────────────────────────┐
│                                                                 │
│  STEP 3: INSTALL ON iPHONE/iPAD                                │
│  ═══════════════════════════════                               │
│                                                                 │
│  1. Open Safari browser (must use Safari!)                     │
│  2. Go to: http://YOUR_IP:5000                                 │
│  3. Tap Share button (□ with ↑)                                │
│  4. Scroll down                                                │
│  5. Tap "Add to Home Screen"                                   │
│  6. Tap "Add"                                                  │
│  7. 🎉 App appears on home screen!                             │
│                                                                 │
└─────────────────────────────────────────────────────────────────┘


┌─────────────────────────────────────────────────────────────────┐
│                                                                 │

│  ═══════════════════════════════════════════════════════       │

│  1. Open Chrome or Edge browser                                │
│  2. Go to: http://localhost:5000                               │
│  3. Look for install icon in address bar (⊕ or 🖥️)            │

═══════════════════════════════════════════════════════════════════
                        ❓ TROUBLESHOOTING
═══════════════════════════════════════════════════════════════════

❌ Install button not showing?
   → Use Chrome/Edge browser
   → Make sure app is running
   → Try clearing browser cache

❌ Can't install on iPhone?
   → Must use Safari browser
   → Use "Add to Home Screen" method
   → iOS doesn't show install prompts

❌ Icons not showing?
   → Run: py generate_icons.py
   → Check static/icons/ folder exists
